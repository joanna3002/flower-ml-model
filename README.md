## Overview
This project demonstrates a simple *supervised machine learning* workflow using the *Iris Flower Dataset*.  
The goal is to build a classification model that predicts the species of an iris flower based on its measurements.

##  Dataset
*Dataset:* Iris Dataset  
*Source:* sklearn.datasets.load_iris() (also publicly available on Kaggle)

*Features (X):*
- Sepal length  
- Sepal width  
- Petal length  
- Petal width  

*Target (y):*  
- 0 = Setosa  
- 1 = Versicolor  
- 2 = Virginica  

Total samples: *150*
